## Story

## Dev Notes

### File Location Decision

**Opzione 1: Standalone file (RACCOMANDATO)**
- Location: `CHAT_EXAMPLES.md` nella root del repository
- Pro: Massima visibilità, facile da trovare
- Con: Altro file in root (ma giustificato per uso frequente)

**Opzione 2: Sezione in docs/README.md**
- Location: `docs/README.md` con sezione "Esempi di Utilizzo"
- Pro: Consolidato con altra documentazione
- Con: Meno visibile per utenti che guardano solo root README

Decisione: Preferire Opzione 1 per visibilità e facilità onboarding.

### Content Guidelines

**Domande Realistiche:**
- Usare terminologia medica appropriata
- Esempi devono riflettere casi d'uso reali
- Evitare domande troppo vaghe o troppo tecniche

**Categorizzazione:**
- **Generali:** Domande overview, principi base, trattamenti comuni
- **Specifiche:** Domande dettagliate su tecniche, controindicazioni, tempi
- **Complesse:** Piani terapeutici, analisi evidenze, confronti approcci

**Disclaimer Medico:**
Includere disclaimer chiaro che risposte sono educative, non consulenza medica.
